# Security Policy

## Supported Versions

Security fixes are applied to the latest `1.x` release line.

## Reporting A Vulnerability

Please do not open public GitHub issues for security reports.

Instead, email [info@ronaldbunk.nl](mailto:info@ronaldbunk.nl) with:

- a clear summary of the issue
- affected versions or commit hashes
- reproduction details or proof of concept
- any suggested remediation, if you have one

We will acknowledge reports as quickly as practical and coordinate remediation
and disclosure once the issue has been validated.
